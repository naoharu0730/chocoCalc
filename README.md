chocoCalc
===

## Description  
チョコットランドのドピ計算したい

## Usage
demo に適当な値を入れれば使えます
## Demo
https://naoharu0730.github.io/chocoCalc/

## Note

- ビタ上昇量
  - ATL/INT/SPD/VIT/LUK：キャラのステ振り * 0.2
  - ALL：キャラのステ振り * 0.1
  - 上昇量は小数点切り捨て
- リキッド上昇量
  - ATK：(ATK - POW) * ((キャラレベル + POW - 100) / 100)
  - DEF：
  - MAT：
  - MDF：

- 魔獣缶
  - 魔獣缶A：POW上昇10、INT減少10、で固定
  - 魔獣缶B：INT上昇10、POW減少10、で固定

- シールの上昇値と減少値
  - 最大の15で固定

- ブラッドスクレイパーの上昇値
  - サブ装備8点着用時の9で固定

- 大天使の加護
- 邪神の呪詛
